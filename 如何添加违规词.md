# 如何添加自定义违规词

## 方法一：直接编辑配置文件（推荐）

### 步骤 1：找到配置文件

配置文件位置：`plugins/AntiSpam/config.yml`

### 步骤 2：编辑违规词列表

打开配置文件，找到 `blocked-words` 部分：

```yaml
anti-spam:
  profanity-filter: true          # 是否启用违规词过滤
  blocked-words:                  # 违规词列表
    - "傻逼"
    - "fuck"
    - "shit"
    - "草泥马"
    - "sb"
    - "nmsl"
```

### 步骤 3：添加你的违规词

在列表中添加新的违规词，每个词占一行，格式为 `- "你的违规词"`：

```yaml
blocked-words:
  - "傻逼"
  - "fuck"
  - "shit"
  - "草泥马"
  - "sb"
  - "nmsl"
  - "你要添加的词1"      # ← 新添加的
  - "你要添加的词2"      # ← 新添加的
  - "badword"           # ← 新添加的
```

### 步骤 4：重新加载配置

在游戏中执行命令：

```
/chat antispam reload
```

或者重启服务器。

### 步骤 5：验证

使用命令查看违规词数量：

```
/chat antispam status
```

输出会显示：
```
- blockedWords: 9 个
```

## 方法二：首次启动自动生成

### 步骤 1：删除旧配置（可选）

如果想重新生成配置文件，删除：
```
plugins/AntiSpam/config.yml
```

### 步骤 2：重启服务器

服务器会自动生成默认配置文件，包含示例违规词。

### 步骤 3：编辑配置

按照方法一的步骤编辑配置文件。

```yaml

  - "傻逼"
  - "傻B"          # 变体
  - "煞笔"         # 变体
  - "nmsl"
  - "NMSL"         # 大写变体
```

## 注意事项

### 1. 格式要求

- 每个违规词必须用引号包围：`"违规词"`
- 每个违规词前面必须有 `- `（短横线+空格）
- 保持正确的缩进（2个空格）

### 2. 正确格式 ✅

```yaml
blocked-words:
  - "word1"
  - "word2"
  - "word3"
```

### 3. 错误格式 ❌

```yaml
# 错误：没有引号
blocked-words:
  - word1
  - word2

# 错误：没有短横线
blocked-words:
  "word1"
  "word2"

# 错误：缩进不对
blocked-words:
- "word1"
- "word2"
```

### 4. 大小写

系统会自动忽略大小写，所以：
- 添加 `"fuck"` 就够了，不需要再添加 `"FUCK"` 或 `"Fuck"`
- 玩家发送 `"FUCK"`, `"fuck"`, `"FuCk"` 都会被拦截

### 5. 包含匹配

系统使用包含匹配，所以：
- 违规词 `"sb"` 会拦截 `"你个sb"`, `"sb啊"`, `"你是sb吗"`
- 要小心过短的违规词可能导致误判

## 测试你的配置

### 1. 添加测试词

```yaml
blocked-words:
  - "测试词"
```

### 2. 重新加载

```
/chat antispam reload
```

### 3. 在游戏中测试

发送消息：`这是一个测试词`

如果看到 `有违规文字`，说明配置成功！

### 4. 查看日志

服务器日志会显示：
```
[AntiSpam] Blocked chat from PlayerName (profanity detected: 测试词)
```

## 常见问题

### Q: 添加了违规词但不生效？

A: 确保：
1. 格式正确（有引号、有短横线、缩进正确）
2. 已执行 `/chat antispam reload`
3. 违规词过滤已启用：`/chat antispam status` 查看

### Q: 如何临时禁用违规词过滤？

A: 使用命令：
```
/chat antispam toggle profanity
```

### Q: 如何清空所有违规词？

A: 编辑配置文件：
```yaml
blocked-words: []
```
然后重新加载。

### Q: 违规词列表有数量限制吗？

A: 理论上没有限制，但建议不超过 100 个以保证性能。

### Q: 可以使用正则表达式吗？

A: 当前版本不支持正则表达式，只支持简单的包含匹配。

## 快速开始

最简单的方式：

1. 打开 `plugins/AntiSpam/config.yml`
2. 在 `blocked-words:` 下面添加你的违规词
3. 执行 `/chat antispam reload`
4. 完成！

示例：
```yaml
blocked-words:
  - "你的违规词1"
  - "你的违规词2"
  - "你的违规词3"
```
