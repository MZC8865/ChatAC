# ChatAntiCheat 功能更新说明

## 已实现的功能

### 1. ✅ 线程安全问题修复

**问题**: `MuteManager.saveMutes()` 在多线程环境下可能导致数据不一致

**解决方案**:
- 在 `MuteManager` 中添加了 `Object lock` 对象
- 所有文件读写操作都使用 `synchronized (lock)` 包裹
- 确保同一时间只有一个线程能访问文件操作

**修改文件**:
- `src/main/java/com/yourname/antispam/managers/MuteManager.java`

### 2. ✅ 定时清理机制

**问题**: `ChatLimiter` 和 `WhisperLimiter` 中的 Map 会无限增长，导致内存泄漏

**解决方案**:
- 在 `ChatLimiter` 和 `WhisperLimiter` 构造函数中启动定时任务
- 每10分钟自动清理离线玩家的数据
- 只保留在线玩家的消息记录

**实现细节**:
```java
// 每10分钟运行一次 (12000 ticks)
plugin.getServer().getScheduler().runTaskTimerAsynchronously(
    plugin, this::cleanupOfflinePlayers, 12000L, 12000L
);
```

**修改文件**:
- `src/main/java/com/yourname/antispam/listeners/ChatLimiter.java`
- `src/main/java/com/yourname/antispam/listeners/WhisperLimiter.java`

### 3. ✅ 白名单系统

**功能**: 允许特定玩家或权限组绕过所有反垃圾检测

**新增文件**:
- `src/main/java/com/yourname/antispam/managers/WhitelistManager.java`

**命令**:
```bash
/chat whitelist add <玩家名>   # 添加玩家到白名单
/chat whitelist del <玩家名>   # 从白名单移除玩家
/chat whitelist list           # 查看白名单列表
```

**特性**:
- 白名单玩家完全绕过所有检测（禁言除外）
- 数据持久化保存到 `whitelist.yml`
- 支持 Tab 补全
- 线程安全的文件操作

**使用场景**:
- 管理员、版主等可信任玩家
- VIP 玩家
- 测试账号

**修改文件**:
- `src/main/java/com/yourname/antispam/AntiSpamPlugin.java` - 初始化管理器
- `src/main/java/com/yourname/antispam/commands/AntiSpamCommand.java` - 添加命令处理
- `src/main/java/com/yourname/antispam/listeners/ChatLimiter.java` - 添加白名单检查
- `src/main/java/com/yourname/antispam/listeners/WhisperLimiter.java` - 添加白名单检查

### 4. ✅ 统计系统

**功能**: 统计每个玩家的消息数量和违规次数

**新增文件**:
- `src/main/java/com/yourname/antispam/managers/StatsManager.java`

**命令**:
```bash
/chat stats <玩家名>   # 查看玩家统计数据
```

**统计内容**:
- 消息数量 - 玩家发送的总消息数
- 违规次数 - 被拦截的消息数
- 违规率 - 违规次数 / 消息数量 * 100%

**特性**:
- 数据持久化保存到 `stats.yml`
- 自动保存（每5分钟）
- 插件关闭时自动保存
- 线程安全的文件操作
- 支持 Tab 补全

**违规类型记录**:
- `muted` - 被禁言
- `profanity` - 违规词
- `spam` - 刷屏（发送过快）
- `similarity` - 消息相似

**修改文件**:
- `src/main/java/com/yourname/antispam/AntiSpamPlugin.java` - 初始化管理器
- `src/main/java/com/yourname/antispam/commands/AntiSpamCommand.java` - 添加命令处理
- `src/main/java/com/yourname/antispam/listeners/ChatLimiter.java` - 记录统计
- `src/main/java/com/yourname/antispam/listeners/WhisperLimiter.java` - 记录统计

## 性能优化

### 1. 内存管理
- 定时清理离线玩家数据，防止内存泄漏
- 使用 `ConcurrentHashMap` 提高并发性能

### 2. 文件IO优化
- 统计数据每5分钟批量保存，减少磁盘IO
- 所有文件操作使用异步任务
- 使用 `synchronized` 确保线程安全

### 3. 定时任务
- 清理任务：每10分钟运行一次
- 统计保存：每5分钟运行一次
- 插件关闭时自动保存所有数据

## 注意事项

### 1. 白名单优先级
- 白名单玩家仍然会被禁言功能影响
- 白名单只绕过反垃圾检测，不绕过禁言

### 2. 统计数据
- 统计数据会持续累积
- 目前没有自动清理机制
- 可以手动删除 `stats.yml` 重置所有统计

### 3. 性能影响
- 定时任务对性能影响极小
- 文件操作都在异步线程中进行
- 适合大型服务器使用
