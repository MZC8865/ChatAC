# 私信检测功能说明

## 功能概述

私信检测功能允许你控制是否对玩家的私聊消息（whisper/私信）进行反垃圾检测。

支持的私信命令：
- `/w <玩家> <消息>`
- `/msg <玩家> <消息>`
- `/tell <玩家> <消息>`
- `/whisper <玩家> <消息>`
- `/m <玩家> <消息>`
- `/t <玩家> <消息>`
- `/pm <玩家> <消息>`

## 功能说明

### 启用状态（默认）
当私信检测启用时：
- 私信会经过所有反垃圾检测（时间间隔、相似度、违规词）
- 违规的私信会被拦截
- 保护玩家免受私信骚扰

### 禁用状态
当私信检测禁用时：
- 私信不会被检测
- 玩家可以自由发送私信
- 只检测公共聊天消息

## 使用命令

### 启用私信检测

```bash
/chat whispercheck on
```

效果：
- 启用私信检测
- 自动保存到配置文件
- 显示：`私信检测已启用并保存到配置文件`

### 禁用私信检测

```bash
/chat whispercheck off
```

效果：
- 禁用私信检测
- 自动保存到配置文件
- 显示：`私信检测已禁用并保存到配置文件`

### 查看当前状态

```bash
/chat whispercheck
```

显示：
```
私信检测状态: 启用
用法:
/chat whispercheck on - 启用私信检测
/chat whispercheck off - 禁用私信检测
```


输出包含：

## Tab 补全支持

- `/chat` + Tab → 显示 `antispam` 和 `whispercheck`
- `/chat whispercheck` + Tab → 显示 `on` 和 `off`

## 配置文件

配置文件位置：`plugins/AntiSpam/config.yml`

```yaml
anti-spam:
  whisper-check: true    # true=检测私信，false=不检测私信
```

### 手动修改配置

1. 编辑 `config.yml`
2. 修改 `whisper-check` 的值
3. 执行 `/chat antispam reload` 重新加载

## 使用场景

### 场景 1：严格管理（推荐）

启用私信检测，防止玩家通过私信发送垃圾消息或违规内容。

```bash
/chat whispercheck on
```

### 场景 2：宽松管理

只检测公共聊天，允许玩家自由私聊。

```bash
/chat whispercheck off
```

### 场景 3：临时调整

在特殊活动期间临时禁用私信检测：

```bash
/chat whispercheck off
# 活动结束后
/chat whispercheck on
```

## 检测范围

### 启用私信检测时，以下内容会被检测：

1. **时间间隔检测**
   - 私信发送过快会被拦截
   - 提示：`请勿刷屏`

2. **相似度检测**
   - 重复或相似的私信会被拦截
   - 提示：`消息过于相似`

3. **违规词过滤**
   - 包含违规词的私信会被拦截
   - 提示：`有违规文字`

### 禁用私信检测时：

- 私信完全不受检测
- 只有公共聊天消息会被检测

## 权限要求

使用 `/chat whispercheck` 命令需要权限：
- `antispam.admin`（默认为 OP）

## 常见问题

### Q: 私信检测默认是启用还是禁用？

A: 默认启用。首次安装插件时，`whisper-check` 默认为 `true`。

### Q: 如何只检测公共聊天，不检测私信？

A: 使用命令 `/chat whispercheck off`

### Q: 修改配置后需要重启服务器吗？

A: 不需要。使用命令修改会自动保存并生效。如果手动编辑配置文件，执行 `/chat antispam reload` 即可。

### Q: 私信检测会影响性能吗？

A: 不会。检测逻辑非常轻量，对性能影响可以忽略不计。

### Q: 可以只禁用私信的某些检测吗？

A: 当前版本不支持。私信检测是全局开关，要么全部检测，要么全部不检测。

### Q: 如何知道私信是否被检测？

A: 查看服务器日志。如果私信被拦截，日志会显示拦截信息。

## 推荐配置

### 公共服务器（推荐启用）

```yaml
whisper-check: true
```

原因：防止玩家通过私信骚扰其他玩家。

### 私人服务器/朋友服（可以禁用）

```yaml
whisper-check: false
```

原因：朋友之间的私聊不需要严格限制。

### 创意服务器（推荐启用）

```yaml
whisper-check: true
```

原因：保护玩家，特别是未成年玩家。
