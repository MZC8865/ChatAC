# 新功能测试指南

## 测试前准备

1. 编译插件：
```bash
./gradlew build
```

2. 将 `build/libs/AntiSpam-3.0.jar` 复制到服务器的 `plugins` 目录

3. 重启服务器

4. 检查日志确认新功能加载：
```
[AntiSpam] Whitelist manager initialized
[AntiSpam] Stats manager initialized
[AntiSpam] Cleanup task started
```

## 测试1：白名单功能

### 步骤1：添加玩家到白名单
```bash
/chat whitelist add TestPlayer
```

**预期结果**：
- 管理员看到：`已将玩家 TestPlayer 添加到白名单`
- TestPlayer 看到：`你已被添加到反垃圾白名单`

### 步骤2：测试白名单玩家绕过检测
TestPlayer 快速连续发送5条相同消息：
```
你好
你好
你好
你好
你好
```

**预期结果**：
- 所有消息都成功发送
- 不会被拦截

### 步骤3：查看白名单
```bash
/chat whitelist list
```

**预期结果**：
```
白名单玩家 (1 个):
- TestPlayer (在线)
```

### 步骤4：移除白名单
```bash
/chat whitelist del TestPlayer
```

**预期结果**：
- 管理员看到：`已将玩家 TestPlayer 从白名单移除`
- TestPlayer 看到：`你已被从反垃圾白名单移除`

### 步骤5：验证移除后的检测
TestPlayer 再次快速发送相同消息：
```
你好
你好
```

**预期结果**：
- 第二条消息被拦截
- TestPlayer 看到：`消息过于相似`

### 步骤6：测试Tab补全
```bash
/chat whitelist <Tab>
  → 显示: add, del, list

/chat whitelist add <Tab>
  → 显示在线玩家列表
```

## 常见问题排查

### 问题1：白名单不生效
**检查**：
1. 玩家是否真的在白名单中：`/chat whitelist list`
2. 是否重新加载了配置
3. 查看日志是否有错误

### 问题2：统计数据不准确
**检查**：
1. 是否等待了5分钟让数据保存
2. 查看 `stats.yml` 文件内容
3. 重启服务器后再次查看

### 问题3：清理任务不运行
**检查**：
1. 查看日志是否有清理消息
2. 确认等待时间足够（10分钟）
3. 检查是否有错误日志

### 问题4：文件损坏
**解决**：
1. 删除损坏的文件
2. 重启服务器
3. 文件会自动重新生成
