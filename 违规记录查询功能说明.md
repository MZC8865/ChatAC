# 违规记录查询功能说明

## 功能概述
新增了 `/chat stats <玩家名> list [时间]` 命令，可以查看玩家在指定时间内的所有违规记录。

## 命令格式
```
/chat stats <玩家名> list [时间]
```

## 参数说明
- `<玩家名>`: 必填，要查询的玩家名称（必须在线）
- `list`: 必填，表示查看违规记录列表
- `[时间]`: 可选，查询的时间范围，默认为 5 分钟

## 时间格式
支持以下时间单位：
- `s` 或 `秒`: 秒
- `m` 或 `分` 或 `分钟`: 分钟
- `h` 或 `时` 或 `小时`: 小时
- `d` 或 `天`: 天

示例：
- `30s` - 30秒
- `5m` - 5分钟
- `10m` - 10分钟
- `1h` - 1小时
- `2h` - 2小时
- `1d` - 1天

## 使用示例

### 1. 查看玩家最近5分钟的违规记录（默认）
```
/chat stats maozhichen list
```

### 2. 查看玩家最近10分钟的违规记录
```
/chat stats maozhichen list 10m
```

### 3. 查看玩家最近1小时的违规记录
```
/chat stats maozhichen list 1h
```

### 4. 查看玩家最近30秒的违规记录
```
/chat stats maozhichen list 30s
```

## 输出格式
命令会显示以下信息：
- 标题：玩家名称和查询时间范围
- 记录总数
- 每条违规记录包含：
  - **时间**（灰色）: 违规发生的时间 (HH:mm:ss 格式)
  - **原因**（红色）: 违规类型（spam/similarity/profanity/muted）
  - **消息内容**（黄色）: 触发违规的消息内容

### 输出示例
```
=== maozhichen 的违规记录（过去 5 分钟）===
共 3 条记录：
[20:15:32] spam: 你好
[20:15:35] similarity: 你好啊
[20:15:40] profanity: 违规词汇
```

## 违规类型说明
- `spam`: 刷屏（消息发送过快）
- `similarity`: 相似消息（消息内容过于相似）
- `profanity`: 违规词汇（包含被屏蔽的词汇）
- `muted`: 禁言状态下发送消息

## 权限要求
需要 `antispam.admin` 权限才能使用此命令。

## 数据存储
- 每个玩家最多保存最近 100 条违规记录
- 违规记录会自动保存到 `plugins/ChatAC/stats.yml` 文件
- 服务器重启后数据不会丢失

## Tab 补全支持
命令支持 Tab 键自动补全：
- `/chat stats <Tab>` - 显示在线玩家列表
- `/chat stats 玩家名 <Tab>` - 显示 "list" 选项
- `/chat stats 玩家名 list <Tab>` - 显示常用时间格式（5m, 10m, 30m, 1h, 2h, 1d）

## 测试步骤

### 1. 准备测试环境
```
# 启动服务器
./gradlew runServer
```

### 2. 触发一些违规行为
在游戏中快速发送多条消息：
```
你好
你好
你好
```

### 3. 查看违规记录
```
/chat stats maozhichen list
```

### 4. 测试不同时间范围
```
/chat stats maozhichen list 1m
/chat stats maozhichen list 10m
/chat stats maozhichen list 1h
```

### 5. 验证颜色显示
确认输出中：
- 时间显示为灰色
- 违规原因显示为红色
- 消息内容显示为黄色

## 注意事项
1. 只能查询在线玩家的违规记录
2. 如果玩家在指定时间内没有违规记录，会显示提示信息
3. 如果玩家没有任何统计数据，会显示"没有统计数据"
4. 时间参数如果无效，会显示错误提示和正确的格式说明
