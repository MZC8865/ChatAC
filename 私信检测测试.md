# 私信检测功能测试指南

## 快速测试步骤

### 准备工作

1. 确保插件已安装：`plugins/AntiSpam-3.0.jar`
2. 重启服务器
3. 确认插件加载成功：
   ```
   [AntiSpam] AntiSpam enabled
   [AntiSpam] Registered chat and whisper listeners
   ```

### 测试 1：基本私信检测

**目标**：验证私信检测功能是否工作

**步骤**：
1. 确保私信检测已启用：
   ```
   /chat whispercheck on
   ```

2. 玩家A发送私信给玩家B：
   ```
   /w PlayerB 测试消息
   ```

3. 立即再发一次相同的消息：
   ```
   /w PlayerB 测试消息
   ```

**预期结果**：
- 第一条消息正常发送
- 第二条消息被拦截
- 玩家A看到：`消息过于相似`（红色）
- 服务器日志：`[AntiSpam] Blocked whisper from PlayerA (similar message, similarity=100.00%)`

### 测试 2：违规词过滤

**目标**：验证私信中的违规词是否被拦截

**步骤**：
1. 确保私信检测和违规词过滤都已启用：
   ```
   /chat whispercheck on
   /chat antispam toggle profanity  # 如果显示"禁用"，再执行一次启用
   ```

2. 玩家A发送包含违规词的私信：
   ```
   /w PlayerB 你是个傻逼
   ```

**预期结果**：
- 消息被拦截
- 玩家A看到：`有违规文字`（红色）
- 服务器日志：`[AntiSpam] Blocked whisper from PlayerA (profanity detected: 傻逼)`

### 测试 3：快速发送检测

**目标**：验证快速发送私信是否被拦截

**步骤**：
1. 确保私信检测已启用：
   ```
   /chat whispercheck on
   ```

2. 玩家A快速连续发送两条不同的私信：
   ```
   /w PlayerB 消息1
   /w PlayerB 消息2
   ```
   （尽可能快地输入）

**预期结果**：
- 第一条消息正常发送
- 第二条消息被拦截（如果间隔小于20ms）
- 玩家A看到：`请勿刷屏`（红色）
- 服务器日志：`[AntiSpam] Blocked whisper from PlayerA (sent too fast, gap=Xms)`

### 测试 4：禁用私信检测

**目标**：验证禁用后私信不被检测

**步骤**：
1. 禁用私信检测：
   ```
   /chat whispercheck off
   ```

2. 玩家A发送相同的私信多次：
   ```
   /w PlayerB 测试
   /w PlayerB 测试
   /w PlayerB 测试
   ```

**预期结果**：
- 所有消息都正常发送
- 没有任何拦截
- 服务器日志中没有拦截记录

### 测试 5：不同私信命令

**目标**：验证所有私信命令都被支持

**步骤**：
1. 启用私信检测：
   ```
   /chat whispercheck on
   ```

2. 测试不同的私信命令（每次发送相同消息）：
   ```
   /w PlayerB 测试
   /w PlayerB 测试
   
   /msg PlayerB 测试
   /msg PlayerB 测试
   
   /tell PlayerB 测试
   /tell PlayerB 测试
   
   /m PlayerB 测试
   /m PlayerB 测试
   ```

**预期结果**：
- 每组命令的第二条消息都被拦截
- 显示：`消息过于相似`

### 测试 6：配置持久化

**目标**：验证配置是否保存

**步骤**：
1. 禁用私信检测：
   ```
   /chat whispercheck off
   ```

2. 查看配置文件 `plugins/AntiSpam/config.yml`：
   ```yaml
   whisper-check: false
   ```

3. 重启服务器

4. 检查配置是否保持：
   ```
   /chat whispercheck
   ```

**预期结果**：
- 配置文件中 `whisper-check: false`
- 重启后状态仍然是"禁用"

### 测试 7：状态查看

**目标**：验证状态命令是否正确显示

**步骤**：
1. 查看总状态：
   ```
   /chat antispam status
   ```

**预期结果**：
```
AntiSpam Status:
- delayMs: 20 ms
- similarityCheck: 启用
- similarityThreshold: 80.00%
- minLengthForCheck: 3
- profanityFilter: 启用
- blockedWords: 6 个
- whisperCheck: 启用    ← 显示私信检测状态
```

## 常见问题排查

### 问题 1：私信没有被检测

**检查清单**：
1. 确认私信检测已启用：`/chat whispercheck`
2. 查看服务器日志是否有加载信息：`Registered chat and whisper listeners`
3. 确认使用的是支持的私信命令（/w, /msg, /tell 等）
4. 检查是否有其他插件拦截了命令

### 问题 2：所有私信都被拦截

**可能原因**：
- 延迟设置过大：`/chat antispam delay 20`
- 相似度阈值过低：`/chat antispam similarity 0.8`

### 问题 3：配置不生效

**解决方法**：
1. 使用命令修改配置（会自动保存）
2. 或手动编辑配置文件后执行：`/chat antispam reload`
