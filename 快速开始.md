# AntiSpam 快速开始指南

## 5分钟上手

### 1️⃣ 安装插件

1. 下载 `AntiSpam-3.0.jar`
2. 放入 `plugins` 文件夹
3. 重启服务器

### 2️⃣ 验证安装

查看服务器日志，应该看到：

```
[AntiSpam] AntiSpam enabled
[AntiSpam] Registered chat and whisper listeners
[AntiSpam] Mute manager initialized
```

### 3️⃣ 基本使用

#### 查看当前配置

```bash
/chat antispam status
```

#### 添加违规词

编辑 `plugins/AntiSpam/config.yml`：

```yaml
blocked-words:
  - "你的违规词1"
  - "你的违规词2"
```

重新加载：

```bash
/chat antispam reload
```

#### 禁言违规玩家

```bash
/chat ban Player 5m    # 禁言5分钟
/chat unban Player     # 解除禁言
```

## 常用命令速查

| 命令 | 说明 |
|------|------|
| `/chat antispam status` | 查看配置 |
| `/chat antispam delay 50` | 设置间隔50ms |
| `/chat antispam similarity 0.8` | 设置相似度80% |
| `/chat whispercheck on` | 启用私信检测 |
| `/chat whispercheck off` | 禁用私信检测 |
| `/chat ban Player 5m` | 禁言5分钟 |
| `/chat unban Player` | 解除禁言 |

## 推荐配置

### 公共服务器（严格）

```yaml
anti-spam:
  delay-ms: 50
  similarity-threshold: 0.8
  profanity-filter: true
  whisper-check: true
```

### 私人服务器（宽松）

```yaml
anti-spam:
  delay-ms: 20
  similarity-threshold: 0.9
  profanity-filter: true
  whisper-check: false
```

## 测试功能

### 测试反垃圾

1. 快速发送两条相同消息
2. 第二条应该被拦截
3. 显示：`消息过于相似`

### 测试违规词

1. 发送包含违规词的消息
2. 应该被拦截
3. 显示：`有违规文字`

### 测试禁言

1. 执行：`/chat ban TestPlayer 1m`
2. TestPlayer 尝试发言
3. 显示：`你已被禁言，剩余时间：XX秒`

## 常见问题

**Q: 插件不工作？**
- 检查 Java 版本是否为 21+
- 检查服务端是否为 Paper 1.21+
- 查看服务器日志是否有错误

**Q: 如何调整严格程度？**
- 更严格：`/chat antispam similarity 0.5`
- 更宽松：`/chat antispam similarity 0.95`

**Q: 配置修改后不生效？**
- 执行：`/chat antispam reload`
